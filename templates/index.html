Welcome to the API of the Restaurant Menu. Below are the available routes to interact with the API.

Available Endpoints:

  • Get all menu items:
    GET /menu
    Returns a list of all menu items in JSON format.

  • Get a specific menu item by ID:
    GET /menu/<int:item_id>
    Returns a specific menu item by its ID.

Response Example:


GET /menu/1
Response:
{
    "id": 1,
    "name": "Tacos al Pastor",
    "price": 25.00,
    "description": "Tacos con carne de cerdo adobada, piÃ±a, cilantro y cebolla"
}

Error Handling:

  • Item not found:
    If the requested menu item is not found, the API will return:
    { "error": "Item not found" } with a 404 status code.

  • File or JSON errors:
    In case of an issue with the menu file, the API will return:
    { "error": "File not found" } or { "error": "Error decoding JSON" } with a 500 status code.
